Manual de Configuração Para Desenvolvimento

#. Instalar vms, softwares e ferramentas:
	Java JDK			1.7 (7u60)
	Eclipse 			Kepler ou STS (SpringToolSuite)
	Postgres 			postgresql-9.3.4-1 64bits
	Git					1.8.3
	Maven 				3.0.X
	Tomcat 				7.X
	pgAdmin 			3
	Sqldeveloper 		4.X
	Notepad++ 			último

#. Baixar código fonte e dependências
	Baixar código fonte a partir do repositório git da delogic em:
		ssh://git@git.delogic.com.br:7999/eg/leitoor-academico.git
		
	Instalar e configurar o maven apontando para o servidor nexus da Delogic de acordo com o arquivo:
		settings.xml (adicionar apenas o mirror se desejar)

#. Configurar recursos para desenvolvimento
	Instalar e configurar o banco de dados postgres, na máquina local:
		Porta: 5432
		Usuário: postgres
		Senha: manager
		
	Instalar e configurar o tomcat dentro do eclipse e configurar:
		Configurar o tomcat que encontra-se na raiz do projeto em \tomcat dentro do Eclipse
		Dentro do Eclipse
			Abrir o arquivo server.xml da pasta Servers
				Alterar o parâmetro ${shutdown.porta} para 8005
				Alterar o parâmetro ${porta} para 8080 (ou outra porta que esteja desbloqueada)

		Adicione a aplicação leitoor ao tomcat e inicie pela primeira vez. Isso irá gerar erros mas irá criar a configuração inicial do tomcat para o próximo passo.
		Abrir o menu de configuracao em Run / Run Configurations
			Na aba Arguments, dentro do textarea VM Arguments, incluir ao final os parâmetros
				-Dspring.profiles.active=DEV
				-DJDBC_CONNECTION_STRING="jdbc:postgresql://localhost:5432/leitoor?user=postgres&password=manager"
				-Dfile.encoding=UTF-8 -Xms128m -Xmx512m -XX:PermSize=64m -XX:MaxPermSize=256m
					
#. Inicializar aplicação
	Para inicializar o banco deve-se executar o teste GerarBanco
		mvn test -Dtest=GerarBanco (ou via JUnit)
		
	Ao inicializar a aplicação o sistema irá: 
		Conectar ao banco de dados criado anteriormente
			Criar estrutura (tabelas, índices, sequences, etc)
			Popular base
